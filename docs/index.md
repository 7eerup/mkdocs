# MkDocs

For full documentation visit [mkdocs.org](https://www.mkdocs.org).

For Material for MkDocs visit [Material for MkDocs](https://squidfunk.github.io/mkdocs-material).

## Commands

* `python -m venv .venv` - Create a Python virtual environment named `.venv`.
* `source .venv/bin/activate` - Activate the Python virtual environment.
* `pip install mkdocs-material` - Install MkDocs Material theme.

* `mkdocs new [dir-name]` - Create a new project.
* `mkdocs serve --livereload` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs -h` - Print help message and exit.

* `mkdocs gh-deploy` - Build and deploy the MkDocs site to GitHub Pages.
